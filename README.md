# 📚 Portal do Aluno SENAI

Projeto desenvolvido para a *Situação de Aprendizagem Desafiadora 2*, com o objetivo de criar uma interface web moderna, responsiva e acessível para o cadastro de alunos em cursos do SENAI.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap 5 (CDN)
* SweetAlert2 (CDN)
* VLibras

---

## ✨ Funcionalidades

* Cabeçalho com identidade visual do Portal do Aluno SENAI;
* Menu de navegação responsivo utilizando Bootstrap;
* Banner principal com botão *"Realizar Cadastro"*;
* Formulário de cadastro contendo:

  * Nome Completo;
  * E-mail;
  * Telefone;
  * Curso desejado;
  * Turno;
  * Aceite dos termos;
* Validação dos campos obrigatórios;
* Mensagens interativas utilizando SweetAlert2;
* Recursos de acessibilidade:

  * Plugin VLibras;
  * Botão de Alto Contraste;
  * Aumentar e diminuir tamanho da fonte;
  * Estrutura HTML semântica;
* Layout responsivo para computadores, tablets e smartphones.

## ⚙️ Como Executar

1. Clone o repositório:

bash
git clone https://github.com/seu-usuario/portal-aluno-senai.git


2. Entre na pasta do projeto:

bash
cd portal-aluno-senai


3. Abra o arquivo index.html em seu navegador.

> Não é necessário instalar dependências, pois Bootstrap e SweetAlert2 são carregados via CDN.
